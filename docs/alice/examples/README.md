# Alice examples

These are *aspirational* `.alice` programs used to drive language design.

| Example | Concepts Demonstrated |
|---------|----------------------|
| [01_types.alice](01_types.alice) | Record props, tuple props, unit props, enum props, const declarations |
| [02_movement.alice](02_movement.alice) | Query syntax, `in Update` schedule, `with` clause, `update` blocks |
| [03_bounds_despawn.alice](03_bounds_despawn.alice) | `Id` keyword, `filter` clause, `despawn` statement |
| [04_collision.alice](04_collision.alice) | `join` for pair iteration, `unique` modifier, built-in functions |

See [spec.md](../spec.md) for the complete language specification.
